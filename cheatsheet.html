Unit 1: Computers

Definitions and Terms

  • Computer - Something that has inputs, outputs, and storage
  • Processor (CPU) - This is the brain of the computer that does all of the calculations
  • Motherboard - Connects all elements to each other, holds USB and expansion card ports, determines size of computer.
  • Case - Holds all components, mainly cosmetic.
  • Graphics Card (GPU) - Processor designed to handle graphics, you hook up the monitor to it.
  • RAM - Stands for random access memory, can buy more of it for the computer, think "short term" memory.
  • Hard Drive - Stores all data, "long term" memory. Includes SSD's (solid state drives) that are faster.
  • Optical Drive - Allows the computer to read CD's and other things.
  • Power Supply - Directs electricity to other components. More speed = more power needed.
  • Inputs - data being put into the computer by peripherals
  • Outputs - The 'feedback' that you are receiving from the computer

Facts from the Test

  • A computer is: Electronic device that stores information
  • What information is held in RAM?: What is on the screen
  • Is a camera a computer?: Yes because there is a input and there is a output, it also has storage.

Unit 2: Data

Definitions and Terms

  • Analog - infinite amount of data, rounded waves, can be sampled, represented as an analogy
  • Digital - finite amount of data, square waves, can be packaged as binary bits

Test Questions

  • All digital data is: Quantized

Unit 3: Binary and Algorithms

Definitions and Terms

  • Algorithm - An algorithm is a series of logical steps to solve a problem. These steps are machine readable.
  • Binary - Base 2 system, 1's represent "on" while 0's represent "off"
  • Decimal - Base 10 system, uses numbers like 223 or 563
  • Greedy Algorithm - An algorithm that takes the extremes first then the middle values
  • Linear Search - Goes down the list until required element is found
  • Binary Search - Needs a sorted list , takes a element and cuts the list in half depending on relative value
  • Processing Time - Number of comparisons made or steps taken to get the desired result

Test Questions

  • Example of a non-computer based algorithm: making a sandwich
